# Carbon Mesh
Library that handles serialization of 3D mesh, skeleton and animation data. It also provides basic
animation runtime support.

## 🛠️ Building

Building can be done using the usual CMake work flow.

By default, cmake will only build the library, without tools (cmfprocessor, viewer). To build tools as well,
configure the project with ``-DBUILD_TOOLS=ON``.
